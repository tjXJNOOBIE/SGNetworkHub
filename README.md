# SGNetworkHub

Minecraft network hub plugin for a Survival Games-era server network. The project provides player-facing hub UI, server navigation, gadgets, scoreboard behavior, and utility handling for a lobby server.

## What It Does

- Displays hub menus for gadgets and server selection.
- Handles player lobby events and hub item behavior.
- Provides lobby scoreboard rendering.
- Includes utility code for item creation and vanish behavior.
- Serves as the entry-point hub for routing players into network game servers.

## Repository Layout

- `src/me/skinlesspotato/hub/gui/` - gadget and server-selection GUIs.
- `src/me/skinlesspotato/hub/listener/` - player lobby event handling.
- `src/me/skinlesspotato/hub/scoreboard/` - lobby scoreboard code.
- `src/me/skinlesspotato/hub/util/` - item and vanish utilities.
- `src/me/skinlesspotato/hub/Hub.java` - plugin entry point.

## Tech Stack

- Java
- Bukkit/Spigot-style plugin APIs
- Single-server lobby/hub plugin architecture

## Status

Archived Minecraft network hub project. The repo is useful as a historical reference for lobby systems around older minigame networks.
